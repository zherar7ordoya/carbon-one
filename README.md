# Carbon One

A light theme built on the principles of IBM’s Carbon palette and Atom’s One Light schema

![Screenshot](https://github.com/zherar7ordoya/Carbon-One/raw/HEAD/assets/capture.png)

> ℹ️ The name **_Carbon One_** honors the work behind IBM’s Carbon Design system and Atom’s One Light theme.

This project is not a reinterpretation or a series —it is a single, finished experiment in light, designed to achieve a balanced state between accessibility and visual ease.

> ℹ️ The HAL-9000 console icon was chosen to echo that sense of precision and restraint.
