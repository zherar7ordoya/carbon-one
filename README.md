# Carbon One

A light theme for VS Code designed for clarity, contrast, and long-term visual comfort.

Based on IBM Carbon Design System. Inspired by Atom One Light.

![Screenshot](https://github.com/zherar7ordoya/Carbon-One/raw/HEAD/assets/capture.png)

## Features

- Soft gray background optimized for real-world lighting
- Language-agnostic syntax highlighting
- WCAG-aligned contrast ratios
- Reduced visual fatigue for extended sessions

## Installation

1. Open **Extensions** in VS Code (`Ctrl+Shift+X`)
2. Search for `Carbon One`
3. Click **Install**
4. Select the theme: `Preferences > Color Theme > Carbon One`
